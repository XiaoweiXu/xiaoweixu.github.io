安装GPU驱动，CUDA和cuDNN

0. 删除卸载用命令

# sudo rm /etc/apt/sources.list.d/cuda*

# sudo apt-get --purge remove "*cublas*" "cuda*" "nsight*"

# sudo apt-get --purge remove "*nvidia*"

# sudo apt-get autoremove

# sudo apt-get autoclean

# sudo rm -rf /usr/local/cuda*

1. 安装NVIDIA显卡驱动

确定PU型号：ubuntu-drivers devices 　

官网查询适合你的GPU的最新驱动版本: www.nvidia.com/Download/index.aspx

下载GPU驱动，并安装: sh cuda_****.run

重启: sudo reboot

查看GPU信息: nvidia-smi

每隔0.5秒执行一次nvidia-smi: watch -n 0.5 nvidia-smi

2. 安装CUDA

确认可以安装的CUDA版本: https://docs.nvidia.com/deploy/cuda-compatibility/index.html#binary-compatibility__table-toolkit-driver

确认PyTorch官网下有CUDA版本适配的信息：https:pytorch.org/
